# Учёт финансов (Finance Tracker)

Десктопное приложение для учёта личных доходов и расходов, написанное на Qt 6 / C++.

## Требования

- Qt 6.2+ с модулями:
  - Qt Widgets
  - Qt Sql
- CMake 3.16+
- Компилятор с поддержкой C++17 (GCC 9+, Clang 10+, MSVC 2019+)

## Сборка
Сборка осуществляется из QtCreator.

Для Windows:
Из под консоли qt в папке сборки
```
windeployqt FinanceTracker.exe
```

## Структура данных

База данных SQLite создаётся автоматически в папке `data/finance.db` рядом с исполняемым файлом.
